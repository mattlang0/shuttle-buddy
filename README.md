# Shuttle Buddy

Shuttle Buddy will organize how your group will get from the put in to the take out in the most economically efficient way and no one gets left behind.

## Run locally

`npx create-expo-app --template`
`npx expo start`


## Deploy
Update `version` in:
 - package.json
 - app.json

Update `versionCode` in:
 - app.json

Run:

`eas build`
